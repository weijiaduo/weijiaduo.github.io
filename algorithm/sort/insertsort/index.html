插入排序

一、算法描述

1.1 核心思想

  • 数据分为已排序区间和未排序区间
  • 插入都是从未排序区间取出元素，插入到已排序区间合适的位置中
  • 插入已排序区间时，同时要保证已排序区间的有序性

1.2 细节解释

比如说，数组 [2, 7, 1, 4, 3, 6, 5]。
