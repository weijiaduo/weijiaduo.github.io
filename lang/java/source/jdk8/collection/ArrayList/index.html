扩容可以分为3步：

  1. 正常情况下，每次扩容增长当前数组容量的 1/2。
  2. 扩容1/2后仍小于最小容量，则使用最小容量；
  3. 若最小容量值大于数组最大值，则使用大数组容量。

2.3 集合操作

集合的操作，不外乎增删查改，由于底层是数组实现，因此对于查询和更新，可以直接根据数组的索引来访问即可，很简单（当然，这中间还会进行索引范围的检查，但也不是难事）。
