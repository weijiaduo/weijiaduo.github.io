3. 总结

总的来说，@Enumerated 注解是用来标识枚举字段是按照什么方式来持久化的，也就是保存到数据库的数据类型：

  1. @Enumerated(EnumType.ORDINAL) 就是保存枚举字段的的整数值（序号）；
  2. @Enumerated(EnumType.STRING) 就是保存枚举字段的字符串值。

另外，不指定类型的 @Enumerated 等于 @Enumerated(EnumType.ORDINAL) 。
