905. 按奇偶排序数组

一、题目描述

给定一个非负整数数组 A，返回一个数组，在该数组中， A 的所有偶数元素之后跟着所有奇数元素。

你可以返回满足此条件的任何数组作为答案。

二、解题思路

利用左右指针，左指针放偶数，右指针放奇数。

  • 左指针从 0 ~ n
  • 右指针从 n ~ 0

三、复杂度分析

  • 时间 O(n)
  • 空间 O(n)
