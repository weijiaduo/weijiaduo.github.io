448. 找到所有数组中消失的数字

一、题目描述

给你一个含 n 个整数的数组 nums ，其中 nums[i] 在区间 [1, n] 内。

请你找出所有在 [1, n] 范围内但没有出现在 nums 中的数字，并以数组的形式返回结果。

二、解题思路

把数值 n 放到它应该在的位置 n - 1，最后再遍历一遍数组，如果 nums[i] != i + 1，就表明数字 i + 1 丢失了。

三、复杂度分析

  • 时间 O(n)
  • 空间 O(1)
